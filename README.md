## For running the Api

In the project change directory:

### `cd /api`

then install dependencies:

### `npm install`
 
then for running the project

### `npm start`

## For running the client(front) side:

first change directory:

### `cd /client` 

then install dependencies:

### `npm install`

and run the app:

### `npm run dev`
